# ✈️ Nabi's Travel Plans

我的个人旅行计划网站，使用 GitHub Pages 托管。

## 🌐 访问地址

- 主页：https://nabiwu.github.io/nabi-trip/
- 墨西哥行程：https://nabiwu.github.io/nabi-trip/trips/mexico.html

## 📁 项目结构

```
nabi-trip/
├── index.html              # 主页 - 展示所有旅行计划
├── trips/
│   └── mexico.html         # 墨西哥行程详情页
├── README.md               # 项目说明
└── Mexico_Summary_for_user.html  # 旧版页面（保留）
```

## 🎨 技术栈

- **Tailwind CSS** - 通过 CDN 使用，无需构建步骤
- **纯静态 HTML** - 完全兼容 GitHub Pages
- **响应式设计** - 支持移动端和桌面端

## 🚀 如何添加新的旅行计划

1. 在 `trips/` 目录下创建新的 HTML 文件（例如：`japan.html`）
2. 参考 `trips/mexico.html` 的结构和样式
3. 在 `index.html` 中添加新的旅行计划卡片

## 📝 特点

- ✨ 现代化的 UI 设计
- 📱 完全响应式，移动端友好
- 🎯 清晰的行程信息展示
- 🔗 所有地点都链接到 Google Maps
- ⚡ 快速加载，无需构建工具
- 🌙 深色主题，护眼舒适

直接在浏览器中打开 `index.html` 即可预览，无需任何构建步骤。
